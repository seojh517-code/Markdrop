# Markdrop — 클립보드를 .md 파일로

`Cmd+C`로 복사해 둔 텍스트를, 현재 열려 있는 Finder 폴더에 마크다운(`.md`) 파일로 저장합니다.
Finder 우클릭 메뉴(빠른 동작)에서 한 번에 실행돼요.

## 구성

| 파일 | 설명 |
|------|------|
| `markdrop.sh` | 핵심 스크립트 (실제 동작 로직) |
| `install.sh` | 스크립트를 `~/.markdrop/`에 설치하고, Finder 우클릭용 **빠른 동작(Quick Action)** 을 생성 |

## 설치

```bash
bash install.sh
```

설치되는 것:
- 핵심 스크립트 → `~/.markdrop/markdrop.sh`
- 빠른 동작 → `~/Library/Services/Markdrop.workflow`

## 사용법

1. 아무 데서나 텍스트를 `Cmd+C` 로 복사합니다.
2. md 파일을 만들고 싶은 폴더를 Finder에서 엽니다.
3. 그 폴더 안의 **파일 하나를 우클릭** → 메뉴 맨 아래 **`Markdrop`** 클릭.
4. 파일명 입력 창에서 이름을 정하면, 현재 열린 폴더에 `.md` 파일이 생성됩니다.

### 동작 규칙
- **저장 위치**: 맨 앞에 열려 있는 Finder 창의 폴더. (열린 창이 없으면 데스크탑)
  - 우클릭한 파일이 어떤 것이든 상관없습니다 — 항상 "현재 열린 폴더"에 저장됩니다.
- **파일명**: 실행하면 **항상 파일명 입력 창**이 뜹니다. 기본값이 전체 선택된 상태라 →
  - 그냥 `Enter` → 기본값 그대로 저장
  - 바로 타이핑 → 선택돼 있던 기본값이 지워지고 새 이름으로 저장
  - `취소` → 저장하지 않음
  - 기본값은 **클립보드 첫 줄**에서 자동 생성 (앞쪽 `#`·공백 제거, 최대 50자).
    첫 줄로 이름을 정할 수 없으면 기본값은 `클립보드`.
  - 같은 이름이 있으면 뒤에 시각(`HHMMSS`)을 붙입니다.
- **인코딩**: 항상 UTF-8.

## 알아둘 점 (macOS 26 기준)

- macOS는 Finder **빈 공간** 우클릭 메뉴에는 빠른 동작을 넣어주지 않습니다.
  그래서 폴더 안의 **파일/폴더를 선택해 우클릭**해야 메뉴에 나타납니다.
  (그 폴더가 비어 있다면, 부모 폴더에서 그 폴더 자체를 우클릭해도 됩니다.)
- 처음 실행 시 "자동화 권한" 또는 알림 권한을 한 번 물어볼 수 있습니다. 허용하세요.

## 제거

```bash
rm -rf ~/.markdrop "~/Library/Services/Markdrop.workflow"
```

## 커스터마이즈
`markdrop.sh`를 수정한 뒤 `bash install.sh`를 다시 실행하면 반영됩니다.
- 저장 위치 고정, 파일명 규칙(예: 날짜·시간 접두어), 확장자 등을 바꿀 수 있습니다.
